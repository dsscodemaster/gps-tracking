######################### Description of the version ##########################

This version upgrades V-2.0. So all the features of V-2.0 includes here along with new features those
are implemented here as below-

1. GPS and GPRS network handle error functionality added here with notifier LED.

2. GSM network configuration for covering wide range(GSM & LTE) of bandwidth.

3. Connection checking and auto reconnection tries to network is implemented here on 
disconnection event. Also periodic modem reset algorithm is implemented in case of long time 
network down.
